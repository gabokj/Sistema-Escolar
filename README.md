# 📚 Sistema Escolar

este é um projeto feito após um desafio em sala para um sistema escolar tendo login, cadastro de alunos e cauculadora de média. 
 Podendo listar todos os alunos, excluir e ver a média dos aprovados e reprovados.

 
## ✨ Funcionalidades 
- 👨‍🎓 Cadastrar aluno 
- 🧮 Caucular as médias dos alunos
- 📋 Listar Alunos
- 💻 Design responsivo CSS

## 📁 Estrutura do Projeto

```
📦Sistema_Escolar/
├── templates     # Pasta onde ficara todo o html.
├──├──base.html   # Template base com estrutura comum.
├──├──cadastro.html  # Página de cadastro de usuários.
├──├──caucular.html  # Página de caucular as médias dos alunos.
├──├──index.html     # Página inicial
├──├──lista.html    # Página que exibe os alunos.
├──└──login.html    # Página de login.
├── static        # Arquivos estáticos.
├──└──style.css   # Estilos personalizados para as páginas HTML.
├── requirements.txt  # Lista das dependências do projeto.
└── app.py/           # Código Python da aplicação, controla rotas e lógica.
```
## 🛠️ Tecnologias Utilizadas
